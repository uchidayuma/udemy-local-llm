# データの準備 STEP1：クリーニング

**ノイズ除去・表記統一・検索されやすい言葉への補完**

---

## このステップの目的

ローカルLLMのRAGは「文書の文字列をベクトル（数値）に変換して検索する」仕組みです。
そのため、**人間が読めても機械が探せない書き方**が多いと、正しい文書が検索されなくなります。

クリーニングの目標は「正確に・一貫して・検索されやすく」の3点です。

---

## 1. ノイズ除去

RAG検索の邪魔になる不要な情報を取り除きます。

### 1-1. 社内用の補足メモ・作成者コメントを削除する

```
【削除前】
洗濯容量：12.0kg  ※営業部確認済み（2025/3）
定格電流：13A  ← 展示用モデルは15Aなので注意

【削除後】
洗濯容量：12.0kg
定格電流：13A
```

> **理由：** 「展示用モデル」「営業部確認」などは別の文書の検索結果を汚染します。

---

### 1-2. 重複する情報を整理する

同じ内容が複数の場所に書かれている場合、1箇所にまとめます。

```
【削除前】
乾燥方式：大風量ヒートポンプ乾燥
※ヒートポンプ乾燥を採用（再掲）

【削除後】
乾燥方式：大風量ヒートポンプ乾燥
```

> **理由：** 重複があるとチャンク（検索の最小単位）が膨らみ、関係のない情報も一緒に渡されます。

---

### 1-3. 空行・区切り線の過剰使用を整理する

```
【削除前】
================
　洗濯コース一覧
================

　　標準コース：約32分


　　時短コース：約17分

【削除後】
## 洗濯コース一覧

- 標準：約32分
- 時短：約17分
```

> **理由：** 記号や空白行はベクトル化のノイズになります。見出し（##）に変換することでチャンクの境界も明確になります。

---

## 2. 表記統一

同じ意味なのに書き方がバラバラだと、検索の精度が下がります。以下のルールで統一します。

### 2-1. 単位の書き方

| 項目 | 統一前（混在） | 統一後 |
|------|--------------|--------|
| 重量 | `12kg` / `12.0KG` / `12キログラム` | `12.0キログラム` |
| 寸法 | `645mm` / `645ミリ` / `64.5cm` | `645ミリメートル` |
| 騒音 | `25dB` / `25デシベル` / `25db` | `25デシベル` |
| 消費電力 | `65Wh` / `65ワット時` | `65ワット時` |
| 電流 | `13A` / `13アンペア` | `13アンペア` |

> **ポイント：** RAG文書は「数値＋単位の日本語表記」に統一すると、日本語モデルとの相性が良くなります。

---

### 2-2. 機種名・型番の書き方

```
【統一前】
UD1000S / UD-1000s / UD1000-S / UDー1000S（全角ハイフン）

【統一後】
UD-1000S（半角ハイフン・大文字）
```

> **理由：** 型番は完全一致で検索されることが多いため、表記ゆれが致命的になります。

---

### 2-3. 機能名・製品固有名詞の書き方

メーカー独自の機能名は、正式名称に統一します。

| 統一前 | 統一後 |
|--------|--------|
| プラズマ除菌 / プラズマパワー | プラズマパワーイオン |
| AIエンジン / 打田AI / AI自動制御 | 打田AIエンジン |
| ヒートポンプ乾燥 / HP乾燥 | 大風量ヒートポンプ乾燥 |
| 凍結防止 / 凍結ヒーター | 自動凍結防止モード |

---

### 2-4. 数値の書き方

```
【統一前】
年間電気代：3,038円 / 3038円 / 約3,038円 / 3,038円（税込）

【統一後】
年間電気代目安：約3,038円（31円/キロワット時換算）
```

> **ポイント：** 「約」「目安」「換算条件」を明記することで、AIが数値の性質（概算か実測か）を正しく理解できます。

---

## 3. 検索されやすい言葉への補完

仕様書には書いてあっても、**ユーザーが使う言葉と違う**ことがあります。
RAGはユーザーの質問文と文書の表現が近いほど正確に検索できるため、同義語・関連語を補完します。

### 3-1. 専門用語に読み替え語を追加する

```
【補完前】
乾燥方式：ハイブリッドヒートポンプ乾燥

【補完後】
乾燥方式：ハイブリッドヒートポンプ乾燥（省エネ性能と乾燥能力のバランスに優れる）
```

> **理由：** ユーザーは「省エネ乾燥」「電気代が安い乾燥」と聞くことが多く、「ヒートポンプ」という専門語だけでは検索でヒットしないことがあります。

---

### 3-2. 数値に文脈を添える

```
【補完前】
設置可能な防水パン：内寸奥行510mm以上

【補完後】
設置可能な防水パン：内寸奥行き510ミリメートル以上（スリムライン同等・マンション標準防水パンに対応）
```

---

### 3-3. 対象ユーザーを自然な言葉で追記する

仕様書には書かれていない「こんな人向け」という情報を補完します。

```
【補完前】
モーター：DDインバーターモーター
洗濯容量：14.0kg
乾燥容量：8.0kg

【補完後】
モーター：DDインバーターモーター
洗濯容量：14.0キログラム（全機種中最大・5人以上大家族・民泊向け）
乾燥容量：8.0キログラム（全機種中最大）
```

> **理由：** ユーザーは「大家族向けの洗濯機を教えて」と聞きます。「14kg」という数値だけでは大家族向けとは検索されません。

---

## チェックリスト

クリーニング作業の完了確認に使用してください。

### ノイズ除去
- [ ] 社内メモ・作成者コメントを削除した
- [ ] 「再掲」「※参考」などの重複情報を整理した
- [ ] 全角記号・過剰な区切り線を削除した

### 表記統一
- [ ] 単位をすべて日本語表記（キログラム・ミリメートル・デシベル）に統一した
- [ ] 型番は半角ハイフン・大文字で統一した（例：UD-1000S）
- [ ] 機能名・製品固有名詞を正式名称に統一した
- [ ] 数値に「約」「目安」「換算条件」を明記した

### キーワード補完
- [ ] 専門用語に分かりやすい読み替え語を追加した
- [ ] 設置寸法に「マンション向け」「防水パン対応」などの文脈を添えた
- [ ] 容量・スペックに対象ユーザー（家族構成・用途）を追記した
