## ファイル構成

各機種は以下の形式でファイルを管理しています。

```
UD-XXXX_spec_base.txt   ← 元データ（Wordで作った平文イメージ）
UD-XXXX_spec_step1.md   ← STEP1：クリーニング（用語の整理・キーワード補完）
UD-XXXX_spec_step2.md   ← STEP2：構造化（見出しによるセクション分け）
UD-XXXX_spec_step3.md   ← STEP3：メタデータ付与（YAMLフロントマター追加）
```

STEP1〜3のファイルセットは、コース演習用の5機種（UD-0700B / UD-1000S / UD-1100T / UD-1100i / UD-1200X）のみ用意しています。
残り25機種はSTEP3（完成形）のみ用意しており、RAGデモのナレッジベースとして利用します。

## 選び方の目安

| こんなとき | おすすめライン |
|-----------|--------------|
| 設置スペースが狭い・防水パンが小さい | **Sライン** |
| とにかく速く洗いたい・泥汚れが多い | **Tライン** |
| スマホで操作・おしゃれな見た目がほしい | **iライン** |
| 最高の機能・静音・省エネがほしい | **Xライン** |
| シンプル・安価・余計な機能は不要 | **Bライン** |
| 高齢の親へのプレゼント | **UD-0800G** |
| ペットの毛が気になる | **UD-0900SP / UD-1100TP / UD-1200XP** |
| 寒冷地（北海道・東北）に住んでいる | **UD-1000SK / UD-1000TC / UD-1200XL** |
